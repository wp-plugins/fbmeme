== Description ==

The FBMeme button easily allows your blog to be shared on Facebook. The button also provides a current count of how many times your story has been shared throughout facebook.

The Plugin is Inspired from the TweetMeme Plugin. It has also used the code that TweetMeme Plugin has.

Features

* Shows the current number of times your post has been shared on Facebook.
* Allows your User to Share a Post on Facebook easily.
* Easily installed.
* Allows you to specify when the button should go (e.g. top or bottom or both)

== Installation ==

Follow the steps below to install the plugin.

1. Upload the fbmeme directory to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings/fbmeme to configure the button
